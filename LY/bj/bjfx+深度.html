data-的详细使用 1. 基本概念 在 HTML 里，你能够给元素添加以 data- 开头的自定义属性，这些属性可以用来存储与元素相关 的额外数据。在 JavaScript 中，通过元素的 dataset 属性就能访问和修改这些自定义数据属性。 2. 自定义数据属性的命名规则 自定义数据属性的命名必须以 data- 开头，后面跟着自定义的名称，名称可以包含字母、数字、连字符 - 等。 在 JavaScript 里，dataset 会把属性名中的连字符分隔的名称转换为驼峰命名法。例如，HTML 中的 data-my-name 在 JavaScript 中通过 dataset.myName 访问。 3.样例
